# Exploratory Data Analysis (EDA) Project

This project was completed as part of the Data Analytics Industrial Training Program by DecodeLabs.

## Objective
Analyze data to identify patterns, trends, and insights.

## Analysis Performed
- Product revenue analysis
- Payment method analysis
- Order status analysis
- Referral source analysis
- Monthly revenue trends
- Descriptive statistics

## Tools Used
- Microsoft Excel
- Pivot Tables

## Outcome
Generated meaningful insights and trends from the dataset using exploratory data analysis techniques.
